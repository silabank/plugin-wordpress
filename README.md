## 🚀 Sobre

O **Sila Pay Checkout** é um gateway de pagamento profissional que integra sua loja WooCommerce à API da **Sila Pay**, permitindo pagamentos rápidos e seguros com:

- 💳 **Cartão de Crédito**
- 📱 **PIX**
- 📄 **Boleto Bancário**

---

## ✨ Funcionalidades

| Recurso | Status | Descrição |
|--------|--------|------------|
| 💳 Cartão de Crédito | ✅ | Parcelamento até 12x, validação em tempo real |
| 📱 PIX | ✅ | QR Code dinâmico, código copia e cola, timer |
| 📄 Boleto | ✅ | Geração automática, PDF, linha digitável |
| 🔒 Webhooks | ✅ | Notificações automáticas de pagamento |
| ⚙️ Configuração automática | ✅ | Sincroniza dados via API |
| 📊 Logs | ✅ | Sistema completo de debug |
| 🎨 Checkout Blocks | ✅ | Compatível com checkout clássico e blocks |

---

## 📋 Requisitos

- WordPress **7.0+**
- WooCommerce **10.0+**
- PHP **7.2+**
- Conta ativa na **Sila Pay**

---

## 🔧 Instalação

### Método 1 — Painel WordPress
1. Vá em **Plugins → Adicionar Novo**
2. Clique em **Enviar Plugin**
  3. Selecione `silapay-payment-gateway-1.1.3.zip`
4. Clique em **Instalar Agora**
5. Ative o plugin
